# Get-SQLiteVersion

## SYNOPSIS

Get SQLite version information

## SYNTAX

```yaml
Get-SQLiteVersion [<CommonParameters>]
```

## DESCRIPTION

This is a simple command to get version information for the System.Data.SQLite assembly.
